# Shift Register Design in Verilog

## Objective
Design a 4-bit shift register demonstrating data storage and shifting operations.

## Features
- Shift Left and Shift Right
- Serial-In Parallel-Out (SIPO)
- Parallel-In Serial-Out (PISO)
- Universal Shift Register

## Tools Used
- Verilog HDL
- ModelSim

## Files
- shift_register.v → Design Code
- shift_register_tb.v → Testbench
- run.do → Simulation script

## Simulation
Run the following command in ModelSim:
do run.do

## Output
Waveforms verified for correct shifting operations.
